# sdv503-a3-hut-booking-13518044
This standalone Node.js command line application addresses a core requirement for the Department of Conservation by replacing error prone paper tracking with an automated booking workflow. The program manages backcountry hut allocations, ensuring that multi-night stays are accurately evaluated night by night against real bunk cpacity constraints to completely prevent double bookings or over allocations. Data automatically persisits into a JSON format upon any modification
#How to Run:
From a fresh terminal run the entry script using Node.js
"Node index.js".
